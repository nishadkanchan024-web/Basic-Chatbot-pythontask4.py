# 🤖 Basic Python Chatbot

A simple rule-based chatbot built using Python. The chatbot responds to basic user inputs like **"hello"**, **"how are you"**, and **"bye"** using predefined replies. This project demonstrates basic Python programming concepts such as **if-elif conditions, functions, loops, and input/output**.

## Features

* Simple rule-based conversation
* Beginner-friendly Python project
* Uses conditional statements and loops
* Easy to modify and expand

## Technologies Used

* Python

## How to Run

1. Clone the repository
2. Open the project folder
3. Run the command:

python chatbot.py

## Example

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
